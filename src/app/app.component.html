String Calculator

Instructions to Test and Run the Environment

  1. Ensure you have Node.js and Angular CLI installed on your machine.
  2. Clone the repository and navigate to the project directory.
  3. Run npm install to install all necessary dependencies.
  4. Run ng serve to start the development server.
  5. Open your browser and navigate to http://localhost:4200.
  6. To run tests, use ng test.

Testing String Calculator

You can test the String Calculator service by running the unit tests provided. The tests cover various scenarios such as handling different delimiters, handling negative numbers, and ignoring numbers greater than 1000.

To run the tests, use the following command:

ng test
